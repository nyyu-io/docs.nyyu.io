## Core Concepts

<div className="grid grid-cols-1 md:grid-cols-2 gap-4 my-6">
  <div className="p-5 rounded-lg border border-border bg-muted/50">
    <div className="text-2xl mb-2">🧩</div>
    <div className="font-bold mb-2">Component</div>
    <div className="text-sm text-muted-foreground">
      Individual avatar accessories like hats or facial features. Each has a <code className="bg-muted px-1 rounded">groupId</code> (category) and <code className="bg-muted px-1 rounded">compId</code> (unique ID).
    </div>
  </div>

  <div className="p-5 rounded-lg border border-border bg-muted/50">
    <div className="text-2xl mb-2">👤</div>
    <div className="font-bold mb-2">Profile</div>
    <div className="text-sm text-muted-foreground">
      Complete avatar built from multiple components, including skills, facts, and personal details.
    </div>
  </div>

  <div className="p-5 rounded-lg border border-border bg-muted/50">
    <div className="text-2xl mb-2">📦</div>
    <div className="font-bold mb-2">AvatarSet</div>
    <div className="text-sm text-muted-foreground">
      Collection of components grouped together with <code className="bg-muted px-1 rounded">groupId</code> and <code className="bg-muted px-1 rounded">compId</code> references.
    </div>
  </div>

  <div className="p-5 rounded-lg border border-border bg-muted/50">
    <div className="text-2xl mb-2">🔤</div>
    <div className="font-bold mb-2">GroupId</div>
    <div className="text-sm text-muted-foreground">
      Category identifier for components (e.g., "hats", "hairColor", "accessories").
    </div>
  </div>
</div>

### AvatarComponent

        <div><code className="bg-muted px-1 rounded">compId</code> - Unique identifier</div>
      </div>
    </div>

    <div className="p-4 rounded-lg bg-background border border-border">
