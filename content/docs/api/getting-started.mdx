## Quick Start

<div className="rounded-xl border border-green-500/30 bg-gradient-to-br from-green-500/10 to-green-500/5 p-6 my-6">
  <div className="flex items-center gap-3 mb-4">

    <div>
      <div className="font-bold text-lg">Start Building in Minutes</div>
      <div className="text-sm text-muted-foreground">Choose your API and begin integration</div>
    </div>
  </div>

  ### NYYU API (GraphQL)

  **Endpoint:** `/graphql`

  ```graphql
  query {
    getUser(id: 1) {
      id
      name
      email
    }
  }
  ```

  ### NYYU PAY (WebSocket)

  **Endpoint:** `wss://api.pay.nyyu.com`

  ```javascript
  const WebSocket = require('ws');
  const ws = new WebSocket('wss://api.pay.nyyu.com');

  ws.on('open', function open() {
    ws.send(JSON.stringify({
      action: 'initiatePayment',
      amount: 1.0,
      currency: 'BTC'
    }));
  });

  ws.on('message', function incoming(data) {
    console.log(data);
  });
  ```

  ### NYYU NODE (WebSocket)

  **Endpoint:** `wss://api.node.nyyu.com`

  ```javascript
  const WebSocket = require('ws');
  const ws = new WebSocket('wss://api.node.nyyu.com');

  ws.on('open', function open() {
    ws.send(JSON.stringify({
      action: 'getBalance',
      currency: 'ETH'
    }));
  });

  ws.on('message', function incoming(data) {
    console.log(data);
  });
  ```

## Authentication and Authorization

<div className="rounded-xl border border-orange-500/30 bg-gradient-to-br from-orange-500/10 to-orange-500/5 p-6 my-6">
  <div className="flex items-center gap-3 mb-4">
    <span className="text-3xl">üîê</span>
    <div>
      <div className="font-bold text-lg">OAuth2 Authentication</div>
      <div className="text-sm text-muted-foreground">Secure your API requests with OAuth2 tokens</div>
    </div>
  </div>

  All NYYU APIs use OAuth2 for authentication and authorization. Include your access token in the `Authorization` header:

  **NYYU API:**
  ```graphql
  {
    "Authorization": "Bearer YOUR_ACCESS_TOKEN"
  }
  ```

  **NYYU PAY and NYYU NODE:**
  ```javascript
  ws.on('open', function open() {
    ws.send(JSON.stringify({
      token: 'YOUR_ACCESS_TOKEN',
      action: 'initiatePayment',
      amount: 1.0,
      currency: 'BTC'
    }));
  });
  ```
</div>
