  <div className="rounded-lg border border-border bg-muted/30 p-5">
    <div className="font-bold mb-2 flex items-center gap-2">
      <span className="text-cyan-500">Q:</span>
      <span>How do I initiate a payment using NYYU PAY?</span>
    </div>
    <div className="text-sm text-muted-foreground pl-6">
      Connect to the WebSocket endpoint <code className="bg-muted px-1 rounded">wss://api.pay.nyyu.io</code> and send a JSON message with payment details including action, amount, and currency. See our payment integration guide for code examples.
    </div>
  </div>

  <div className="rounded-lg border border-border bg-muted/30 p-5">
    <div className="font-bold mb-2 flex items-center gap-2">
      <span className="text-cyan-500">Q:</span>
      <span>How do I get the balance of a cryptocurrency using NYYU NODE?</span>
    </div>
    <div className="text-sm text-muted-foreground pl-6">
      Connect to the WebSocket endpoint and send a message with action "getBalance" and specify the currency (ETH, BTC, etc.). The response will include the current balance.
    </div>
  </div>

  <div className="rounded-lg border border-border bg-muted/30 p-5">
    <div className="font-bold mb-2 flex items-center gap-2">
      <span className="text-cyan-500">Q:</span>
      <span>What is the rate limit for NYYU APIs?</span>
    </div>
    <div className="text-sm text-muted-foreground pl-6">
      Rate limits vary by endpoint and authentication tier. Implement exponential backoff for rate-limited requests and monitor response headers for limit information.
    </div>
  </div>
</div>
