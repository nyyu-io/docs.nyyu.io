## Authentication

<Callout type="warn">
ðŸ”’ **OAuth2 Authentication Required** - All API requests must include a valid OAuth2 token
</Callout>

### Getting Your API Token

<Steps>
<Step>
### Register Your Application
Create an OAuth2 application in your NYYU dashboard
</Step>

<Step>
### Get Credentials
Obtain your `client_id` and `client_secret`
</Step>

<Step>
### Request Token
Exchange credentials for an access token
</Step>

<Step>
### Use Token
Include token in `Authorization` header: `Bearer YOUR_TOKEN`
</Step>
</Steps>
