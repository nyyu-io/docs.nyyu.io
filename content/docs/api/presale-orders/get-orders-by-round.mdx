## Overview

The `getPresaleOrders` query allows administrators to retrieve all orders placed in a specific presale round, providing insights into participation levels and token distribution.

---

## GraphQL Schema

<div className="rounded-xl border border-fuchsia-500/30 bg-gradient-to-br from-fuchsia-500/10 to-purple-500/5 p-6 mb-8">

```graphql
query {
  getPresaleOrders(
    presaleId: Int!
  ): [PreSaleOrder]
}
```

</div>

## Example Usage

### Request

```graphql
query GetRoundOrders {
  getPresaleOrders(presaleId: 12) {
    id
    presaleId
    userId
    ndbAmount
    destination
    extAddr
    status
    createdAt
    updatedAt
  }
}
```
